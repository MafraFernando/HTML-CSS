Estilos Locais / Internos

organização

Ao utilizar os estilos dentro da área head os códigos ficam separados HTMl e CSS se tornando muito mais organizado do que linha a linha o CSS dentro do código HTML

Mudar todos do mesmo

Se eu desejo mudar a cor ou fonte de todos os h1 basta utilizar dentro de head H1 {color: }

Estilos Externos

O mais versátil

As CSS externas são as mais utilizadas, abrimos um arquivo apenas com os estilos e se torna padrão para todas as páginas ao utilizar o link abaixo do title <link rel="stylesheet" href="style.css">. Os estilos seguem a hierarquia Inline>interno>externo
