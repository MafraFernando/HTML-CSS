Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negritousando a tag < b > (não semântica).

Nesta frase, tenho um termo em destaqueusando o < strong > (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálicousando a tag < i > (não semântica).

Nesta frase, temos um termo em ênfase usando a tag < em > (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag < mark > que ficará originalmente em amarelo

E no outro parágrafo, temos outro texto marcado para mostrar que se usa a tag para cada marcação ou um único style para modificar a cor de todas as marcações da mesma página que deve ser utilizado abaixo do title.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequenonesse parágrafo. a tag < big > já caiu em desuso.

Texto deletado (riscado)

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. tag < del >

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. tag < ins >

Existe também o sublinhado com a tag < u > (não semântica)

Texto sobrescrito

Para inserir coisas o tipo x20+3 usando a tag < sup >

Texto subscrito

Para inserir coisas do tipo H2O usando a tag < sub >
