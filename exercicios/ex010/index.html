Usando links internos

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página

Para utilizar uma página dentro do próprio site é necessário criar um novo arquivo HTML dentro da mesma pasta do arquivo index. utulizar < a href="CTRL+SPACE"> localiza o arquivo

Para facilitar os mecanismos de busca é utilizado o rel="prev" e rel="next" para saber qual página é anterior e qual a posterior
